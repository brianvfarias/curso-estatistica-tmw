# Introdução à Estatística

## O que é estatística

Estatística, com "E" maiúsculo, é a área do conhecimento que estuda as formas de sumarizar e entender dados.

Já uma estatísticas, com "e" minúsculo pode ser uma contagem, taxa/proporção ou outro dado que resume ou sumariza uma população/conjunto de dados.

Resumindo: **estatística é um dado que sumariza um conjunto de dados**

## População X Amostra

População: corresponde ao conjunto inteiro de dados que se querem analisar. Exemplos: todo o sangue do corpo, todas as pessoas de uma determinada região, todos os dispositivos de uma determinada marca/modelo.

Infelizmente, em muitos cenários, considerar a população inteira para fazer uma análise é inviável. Dessa maneira, coletar amostras é a aboardagem utilizada.

Amostra: é uma parcela da população. Exemplos: xml de sangue extraídos p/ o um hemograma, algumas pessoas de determinada região, alguns lotes de dispositivos de uma marca/modelo.

Para garantir a confiabilidade e precisão das análises geradas utilizam-se as técnicas de **Amostragem Aleatório** que buscam garantir que as amostras são **amostras represetativas** e **não enviesadas**.

### Amostra

#### Parâmetro x Estimador

Parâmetro: descreve uma característica medida do conjunto de dados completo.

Estimador: estimativa de um parâmetro

| -                     | Parâmetro                       | Estimador             |
| --------------------- | ------------------------------- | --------------------- |
| **conjunto de dados** | População                       | Amostra               |
| **valor**             | Fixo (normalmente desconhecido) | Calculado e aleatório |

Queremos descobrir a renda média das pessoas trabalhadoras do Brasil.
Podemos perguntar para todo mundo?

**População**: Pessoas brasileiras que estão com a carteira de trabalho ativa

**Amostra**: Respondentes de uma enquete de salário no Glassdoor

**Unidade Amostral**: Indivíduo respondente

**Variáveis (ou características) da unidade amostral**: Informações da unidade amostral: gênero, idade, estado de residência, escolaridade, cargo, salário.
