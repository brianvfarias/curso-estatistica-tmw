# Tipos de Variáveis

## Qualitativas

As variáveis qualitivativas são variáveis não numéricas como, por exemplo, cor, gênero, raça, etnia, religião, categoria, cargo etc.

### Variáveis Qualitativas Nominais

São aquelas variáveius qualitativas que não possuem uma ordem, como ,por exemplo, cor e cargo

### Variáveis Qualitativas Ordinais

São varíaveis que possuem uma ordem por trás, como, por exemplo, nível de escolaridade, senioridade e, idealmente, indíces de satisfação de um cliente (NPS). No caso dos **NPS, mesmo que sejam** apresentadas opções **numéricas** para os respondentes, **idealmente esses dados serão interpretados como qualitativos ordinais e não como quantitativos**.

## Quantitativas

As variáveis numéricas, como por exemplo, quantidade de dias desde a última compra, salário, idade etc.

### Variáveis Quantitativas Discretas

Dados em que as casas decimais não são consideradas. Alguns exemplos são idade e quantidade de filhos. (Números inteiros)

### Variáveis Quantitativas Contínuas

Dados em que as casas decimais são consideradas e são relevantes. Alguns exemplos são peso, altura e salário. (Números de ponto flutuante)
