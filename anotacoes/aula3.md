# Tabela de Frequência

## Medidas de Resumo

**Medidas de Resumo** ou **Estatísticas Descritivas** são valores que resumem os conjuntos de dados. Os exemplos mais comuns são Média, Moida e Mediana.

## Tabela de Frequência

Tabelas de frequência são um tipo de Medida de Resumo. Elas servem para exibir quanto cada nível/valor de uma variável ocorre, como esses níveis se comparam entre si e como se comparam com o todo.

Os componentes da Tabela de Frequência serão exemplificados considerando a seguinte tabela como o conjunto de dados original.

| Nome     | Gênero    |
| -------- | --------- |
| Mariana  | Feminino  |
| Tica     | Feminino  |
| Giovana  | N/A       |
| Ariel    | N/A       |
| Mauricio | Masculino |
| Eduardo  | Masculino |
| Henrique | Masculino |

### Frequência Absoluta ($n_i$)

Mostra quantas vezes cada valor/nível aparece dentro do conjunto de dados para uma variável específica.

**Variável/Característica Gênero**

| Nível     | Frequência Absoluta ($n_i$) |
| --------- | --------------------------- |
| Feminino  | 2                           |
| Masculino | 3                           |
| N/A       | 2                           |

### Frequência Relativa ($f_i$)

É a proporção ou taxa que cada nível representa em comparação ao total.

Ela pode ser calculada dividindo o valor de cada nível pelo valor total de Unidades Amostrais da Variável/Conjunto de Dados.

| Nível     | Frequência Absoluta ($n_i$) | Frequência Relativa ($f_i$) |
| --------- | --------------------------- | --------------------------- |
| Feminino  | 2                           | 0,2857                      |
| Masculino | 3                           | 0,4285                      |
| N/A       | 2                           | 0,2857                      |

### Frequência Absoluta Acumulada ($N_i$)

É o acúmulo/soma das frequências absolutas do nível atual com os níveis anteriores.

| Nível     | Frequência Absoluta ($n_i$) | Frequência Absoluta Acumulada ($N_i$) | Frequência Relativa ($f_i$) |
| --------- | --------------------------- | ------------------------------------- | --------------------------- |
| Feminino  | 2                           | 2                                     | 0,2857                      |
| Masculino | 3                           | 5                                     | 0,4285                      |
| N/A       | 2                           | 7                                     | 0,2857                      |

### Frequência Relativa Acumulada ($F_i$)

É o acúmulo/soma das frequências relativas do nível atual com os níveis anteriores.
| Nível | Frequência Absoluta ($n_i$) | Frequência Absoluta Acumulada ($N_i$) | Frequência Relativa ($f_i$) | Frequência Relativa Acumulada ($F_i$) |
| --------- | --------------------------- | ------------------------------------- | --------------------------- | -- |
| Feminino | 2 | 2 | 0,2857 | 0,2847 |
| Masculino | 3 | 5 | 0,4285 | 0,7142 |
| N/A | 2 | 7 | 0,2857 | 1 |
